W3C, HTML, & CSS

W3C

World wide web consortium or W3C is a vendor-neutral organisation created in 1994 that develops common interoperable protocols for the World Wide Web. W3C is primary activity is to develop protocols and guidelines that ensure long-term growth for the web. W3C standards define key parts of what makes the World Wide Web. Represented by the Massachusetts Institute of technology in the United States and a number of international research centers, the World Wide Web consortium provides a variety of services to its member organisations including the following: discussion groups and meetings on issues relating to the evolution of the World Wide Web, repositories of information, reference documents, and code relating to WWW protocols services, and applications.

HTML

HTML stands for Hypertext markup language this is the language that you use to tell a web browser how to make a page look. HTML is what are use to make a title on the website big and bold, change fonts, insert images, and reposition different elements on a webpage.

CSS

Modern modern day website is made up of three files: HTML, JavaScript, and CSS. CSS defines the overall appearance and style of a page. CSS is short for cascading style sheet, it helps determine things like colors, fonts, positioning of certain elements and even animation.
