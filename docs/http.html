HTTP & HTTPS

Hypertext Transfer Protocol (Secure)

When entering a website in that moment your computer starts talking to another computer called a server that is usually thousands of miles away and in milliseconds your computer asks that server for a website and that server starts to talk back to your computer in a language called HTTP. HTTP stands for hypertext transfer protocol which is a language that one computer uses to ask another computer for a document. HTTP manages the sending and receiving of HTML, media files, or anything on the web. HTTPS which stands for hypertext transfer protocol secure, ensures that your HTTP requests are secure and protected such as when logging into a website.
